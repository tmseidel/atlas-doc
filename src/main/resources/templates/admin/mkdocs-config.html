MkDocs Configuration

Configure the Git repository that holds mkdocs.yml, custom CSS, and theme overrides. The Theme Stakeholder interacts with the system exclusively through this repository.

Configuration Repository
For private repos: enter a Gitea personal access token.
How it works

The Build Service clones/pulls this repository before each build. Changes to mkdocs.yml, custom.css, or overrides/ are applied automatically on the next build cycle.
